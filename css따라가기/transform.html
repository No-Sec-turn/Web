transform 변형

transform: ; 변형함수의 종류

  1. translate() 위치 변형
  2. scale() 크기 변형
  3. rotate() 회전 변형
  4. skew() 기울기 변형

transform:translate(); 위치 변형

transform:translateX(800px);
transform:translateY(200px);
transform:translateX(-250px);
transform:translateY(-300px);
transform:translate(900px,-350px);
transform:translate(600px,250px);
transform:translate(100%,100%);
transform:translate();
[항상 center에 위치하기] 1. 상위요소의 설정은 position: relative;로, 중앙정렬의 대상은 position: absolute; top:50%; left: 50%;로 설정 2. 추가로 transform:translate(-50%,-50%);하여 자신의 크기 절반만큼 위로,앞으로 이동 3. 항상 중앙정렬 끝
