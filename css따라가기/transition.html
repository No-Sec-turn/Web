transition 단순한 애니메이션 효과

transition의 세부속성들

  1. transition-property: ; - 효과 줄 속성
  2. transition-duration: ; - 진행 시간
  3. transition-delay: ; - 지연 시간
  4. transition-timing-function: ; - 가속감속 등
    • ease = (기본값) 느리게 시작해서 빨라지다가 천천히 끝남
    • ease-in = 시작을 느리게
    • ease-out = 마지막을 느리게
    • ease-in-out = 시작과 마지막을 같이 느리게
    • linear = 등속
    • cubic-bezier(n,n,n,n) = 베지에함수값으로 지정
      cubic-bezier.com (베지에함수값 자동생성기)
